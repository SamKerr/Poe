# Epic: Hardening and Quality

## Goal

Add lightweight protections and test/docs coverage so the MVP is reliable and harder to pollute.

## Why this exists

Anonymous posting requires basic safeguards to keep data quality acceptable without heavy moderation systems.

## Scope

- Rate limiting and duplicate guards
- Integration and unit tests for main behaviors
- Documentation updates for API contracts and mindful feed model

## Out of scope

- Full moderation platform
- ML spam detection
- Authentication-based abuse controls

## Done when

Core endpoints are guarded by light anti-abuse checks, tested, and clearly documented.
