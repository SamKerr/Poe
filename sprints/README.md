# Sprint Workspace

Planning and delivery tracking for this repo lives in `sprints/`.

Structure:

- `plans/` - active sprint plan files
- `plans/done/` - completed sprint plan files
- `tickets/` - active ticket files for current sprint work
- `tickets/done/` - completed ticket files

Rules:

- Keep only active items in `plans/` and `tickets/`
- Move completed items to the matching `done/` folder
- Keep sprint plans and ticket files aligned
